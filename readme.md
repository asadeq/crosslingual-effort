# Raw Datasets

This repository consists of two datasets:
- Participant Background and Demographics (participant_data.csv)
- Eye gaze data for each visual stimulus and each participant (eyetrack_data.tsv)

These can be opened and examined used either Excel or Python

# Statistical Analysis Script

The data_analyzer.ipnyb contains a Python Jupyter script that was used to generate all the statistics reported in the study (and more that were not included).
